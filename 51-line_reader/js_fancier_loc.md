# fancier line of code

write an advance version of loc.js command line application to count lines of code within a file.

example usage:
```
$ node loc.js test1.js
total lines: 5
code lines: 2
comment lines: 2
blank lines: 1
```

example usage:
```
$ node loc.js test2.js
total lines: 3
